18  Hypothesis Testing and Comparisons

Descriptive statistics and visualizations tell you what your data looks like. Hypothesis testing tells you whether the patterns you see are likely to be meaningful or whether they could have arisen by chance. This chapter introduces a small set of statistical tests — enough to make principled statements about group differences and variable relationships — and focuses on interpreting results in plain language rather than on statistical theory.

The framework is deliberately simple. You will learn to ask three questions: Is there a difference between these groups? How strong is the relationship between these variables? Could what I’m seeing be due to random variation? The tools that answer these questions — t-tests, chi-square tests, and correlation coefficients — cover a large proportion of the exploratory comparisons you will encounter with tabular data.

18.1 Setup

import pandas as pd
import numpy as np
import matplotlib.pyplot as plt
import seaborn as sns
from scipy import stats

pd.set_option('display.max_columns', None)
pd.set_option('display.float_format', '{:.3f}'.format)
sns.set_style('whitegrid')

18.2 Datasets

# Lahman batting — qualified player-seasons with salary
batting = pd.read_csv('Batting.csv')
batting['BA'] = batting['H'] / batting['AB'].replace(0, float('nan'))
batting['TB'] = batting['H'] + 2*batting['2B'] + 3*batting['3B'] + 4*batting['HR']
batting['SLG'] = batting['TB'] / batting['AB'].replace(0, float('nan'))
batting['HBP'] = batting['HBP'].fillna(0)
batting['SF'] = batting['SF'].fillna(0)
batting['OBP'] = (
    (batting['H'] + batting['BB'] + batting['HBP'])
    / (batting['AB'] + batting['BB'] + batting['HBP'] + batting['SF']).replace(0, float('nan'))
)
batting['OPS'] = batting['OBP'] + batting['SLG']
batting['decade'] = (batting['yearID'] // 10) * 10
qualified = batting[batting['AB'] >= 100].copy()

salaries = pd.read_csv('Salaries.csv')
bat_sal = pd.merge(
    qualified, salaries[['playerID', 'yearID', 'salary']],
    on=['playerID', 'yearID'], how='inner'
)
# Cannabis sales
cannabis = pd.read_csv(
    'colorado_cannabis_sales_by_county.csv',
    header=2, skipfooter=8, engine='python', na_values=['NR']
)
cannabis.columns = ['month', 'year', 'county', 'medical', 'retail']
cannabis['county'] = cannabis['county'].str.strip()
for col in ['medical', 'retail']:
    cannabis[col] = pd.to_numeric(
        cannabis[col].str.replace('$', '', regex=False).str.replace(',', '', regex=False),
        errors='coerce'
    )
cannabis = cannabis[cannabis['county'] != 'Total']

18.3 The Logic of Hypothesis Testing

Every hypothesis test follows the same structure:

  1. State the null hypothesis (H₀): There is no difference between the groups, or there is no relationship between the variables. The observed pattern is due to chance.
  2. State the alternative hypothesis (H₁): There is a real difference or relationship.
  3. Compute a test statistic that measures how far the observed data deviates from what the null hypothesis would predict.
  4. Compute a p-value — the probability of seeing a test statistic this extreme (or more extreme) if the null hypothesis were true.
  5. Interpret the result. A small p-value (conventionally below 0.05) means the observed pattern is unlikely to have arisen by chance alone. A large p-value means you do not have enough evidence to reject the null hypothesis.

A p-value does not tell you the probability that the null hypothesis is true. It tells you how surprising your data would be if the null hypothesis were true. This distinction matters: a p-value of 0.03 means “if there were truly no difference, we’d see data this extreme only 3% of the time” — not “there is a 97% chance the difference is real.”

WarningStatistical significance is not practical significance

A large dataset can produce a tiny p-value for a difference that is too small to matter. Always report and interpret the effect size — the magnitude of the difference — alongside the p-value. A statistically significant but tiny difference is not analytically useful.

18.4 Comparing Two Groups: The t-Test

The independent samples t-test asks: Is the mean of one group different from the mean of another group? Use it when you have one numeric outcome and two categorical groups.

18.4.1 Example: Batting average before and after the steroid era

The late 1990s through early 2000s are sometimes called baseball’s “steroid era.” Did batting averages differ between the pre-steroid period (1980–1993) and the steroid era (1994–2004)?

# Filter to each era separately and extract the BA values as a Series
# dropna() removes rows where BA could not be computed (zero at-bats, etc.)
pre_steroid = qualified[
    (qualified['yearID'] >= 1980) & (qualified['yearID'] <= 1993)
]['BA'].dropna()

steroid_era = qualified[
    (qualified['yearID'] >= 1994) & (qualified['yearID'] <= 2004)
]['BA'].dropna()

print(f'Pre-steroid: n={len(pre_steroid)}, mean={pre_steroid.mean():.4f}, std={pre_steroid.std():.4f}')
print(f'Steroid era: n={len(steroid_era)}, mean={steroid_era.mean():.4f}, std={steroid_era.std():.4f}')

Visualize the distributions before testing:

fig, ax = plt.subplots(figsize=(8, 4))
sns.kdeplot(pre_steroid, label='Pre-steroid (1980–93)', fill=True, alpha=0.3, ax=ax)
sns.kdeplot(steroid_era, label='Steroid era (1994–2004)', fill=True, alpha=0.3, ax=ax)
ax.set_xlabel('Batting Average')
ax.set_title('Batting Average Distribution: Pre-Steroid vs. Steroid Era')
ax.legend()
plt.tight_layout()
plt.show()

Now run the t-test:

# ttest_ind: independent-samples t-test (comparing two separate groups)
# Returns (t_statistic, p_value) as a tuple
# A large |t| and small p indicates the means are unlikely to be equal by chance
t_stat, p_value = stats.ttest_ind(pre_steroid, steroid_era)

print(f't-statistic: {t_stat:.3f}')
print(f'p-value: {p_value:.4f}')

18.4.2 Interpreting the result

The mean difference between the two eras can be computed directly:

diff = steroid_era.mean() - pre_steroid.mean()
print(f'Mean difference: {diff:.4f}')

Here is what you can conclude: if the p-value is below 0.05, the difference in mean batting average between the two eras is statistically significant — unlikely to have arisen by chance. The effect size (the mean difference) tells you whether the difference is large enough to be practically meaningful. A difference of 0.005 in batting average is detectable with thousands of player-seasons but may not be substantively interesting. A difference of 0.015 would be more notable.

What you cannot conclude: that steroids caused the difference. These are different groups of players measured in different years with different rules, ballparks, and training methods. The comparison is between-subjects and observational — all the caveats from Chapter 6 apply.

18.4.3 Effect size: Cohen’s d

Cohen’s d measures the difference in means in terms of standard deviations. It provides a standardized measure of effect size that is comparable across different variables and studies.

# Cohen's d = mean difference / pooled standard deviation
# Pooled std averages the variance of both groups, giving equal weight to each
pooled_std = np.sqrt((pre_steroid.std()**2 + steroid_era.std()**2) / 2)
cohens_d = (steroid_era.mean() - pre_steroid.mean()) / pooled_std

# Interpretation guide: |d| < 0.2 = negligible, 0.2–0.5 = small, 0.5–0.8 = medium, > 0.8 = large
print(f"Cohen's d: {cohens_d:.3f}")

Rules of thumb for interpreting Cohen’s d: 0.2 is a small effect, 0.5 is medium, and 0.8 is large. These are guidelines, not thresholds — context determines what counts as meaningful in your domain.

18.5 Comparing Two Related Measurements: The Paired t-Test

A paired t-test is appropriate when you have two measurements from the same entity — for example, medical and retail cannabis sales from the same county in the same month. The question is: Is there a systematic difference between the two measurements?

# Compare medical vs retail sales for county-months where both exist
paired = cannabis[cannabis['medical'].notna() & cannabis['retail'].notna()].copy()

print(f'Paired observations: {len(paired)}')
print(f'Mean medical: ${paired["medical"].mean():,.0f}')
print(f'Mean retail:  ${paired["retail"].mean():,.0f}')

t_stat, p_value = stats.ttest_rel(paired['retail'], paired['medical'])

print(f'\nPaired t-test:')
print(f't-statistic: {t_stat:.3f}')
print(f'p-value: {p_value:.6f}')

Here is what you can conclude: if the p-value is very small, retail sales are systematically different from medical sales across county-months. The direction and magnitude of the difference tell you which is larger and by how much. This is a within-subjects comparison — the same counties are measured on both variables — so many county-level confounds (population, geography) are controlled for.

18.6 Comparing Categorical Variables: The Chi-Square Test

The chi-square test of independence asks: Are two categorical variables associated, or are they independent? Use it when both your variables are categorical.

18.6.1 Example: Has the distribution of home run tiers changed across eras?

# Bin HR into categories
hr_data = qualified[qualified['decade'].isin([1980, 2000])].copy()
hr_data['hr_tier'] = pd.cut(
    hr_data['HR'], bins=[-1, 5, 15, 30, 100],
    labels=['0-5', '6-15', '16-30', '31+']
)
hr_data['era'] = hr_data['decade'].map({1980: '1980s', 2000: '2000s'})

# Build contingency table
contingency = pd.crosstab(hr_data['era'], hr_data['hr_tier'])
print('Contingency table:')
print(contingency)
chi2, p_value, dof, expected = stats.chi2_contingency(contingency)

print(f'\nChi-square statistic: {chi2:.2f}')
print(f'Degrees of freedom: {dof}')
print(f'p-value: {p_value:.6f}')

Here is what you can conclude: if the p-value is small, the distribution of home run tiers differs significantly between the two eras — the two variables (era and HR tier) are not independent. To understand how they differ, compare the observed counts to the expected counts (what you would see if era and HR tier were independent):

expected_df = pd.DataFrame(
    expected, index=contingency.index, columns=contingency.columns
).round(1)

print('Expected counts (if independent):')
print(expected_df)
print('\nObserved - Expected:')
print((contingency - expected_df).round(1))

Positive residuals indicate cells with more observations than expected; negative residuals indicate fewer. This tells you which specific categories drive the overall association.

18.6.2 Effect size: Cramér’s V

Cramér’s V is the chi-square analog of Cohen’s d — a measure of how strongly the two categorical variables are associated, scaled from 0 (independent) to 1 (perfectly associated):

n = contingency.sum().sum()
min_dim = min(contingency.shape) - 1
cramers_v = np.sqrt(chi2 / (n * min_dim))

print(f"Cramér's V: {cramers_v:.3f}")

18.7 Measuring Linear Relationships: Correlation

The Pearson correlation coefficient (r) measures the strength and direction of a linear relationship between two numeric variables. It ranges from -1 (perfect negative linear relationship) to +1 (perfect positive linear relationship), with 0 indicating no linear relationship.

18.7.1 Example: Salary and performance

# Filter to a single recent season with enough data
season_2015 = bat_sal[bat_sal['yearID'] == 2015].dropna(subset=['salary', 'OPS'])

r, p_value = stats.pearsonr(season_2015['salary'], season_2015['OPS'])

print(f'Pearson r: {r:.3f}')
print(f'p-value: {p_value:.4f}')
print(f'n: {len(season_2015)}')

Visualize alongside the statistic:

fig, ax = plt.subplots(figsize=(8, 5))
ax.scatter(season_2015['salary'] / 1e6, season_2015['OPS'], alpha=0.4, s=20)
ax.set_xlabel('Salary (millions $)')
ax.set_ylabel('OPS')
ax.set_title(f'Salary vs. OPS (2015) — r = {r:.3f}, p = {p_value:.4f}')
plt.tight_layout()
plt.show()

Here is what you can conclude: the correlation coefficient tells you the strength and direction of the linear relationship. A positive r means higher salary is associated with higher OPS. The p-value tells you whether this correlation is statistically significant. The magnitude of r tells you how tightly the points cluster around a line — an r of 0.3 indicates a weak-to-moderate positive relationship, meaning salary explains about 9% (r² = 0.09) of the variation in OPS.

What you cannot conclude: that paying a player more would improve their performance. As discussed in Chapter 6, the causal arrow is ambiguous, and the relationship is confounded by age, experience, and roster construction.

18.7.2 Spearman correlation for non-linear monotonic relationships

Pearson’s r measures linear relationships. If the relationship is monotonic but curved (both variables increase together, but not in a straight line), Spearman’s rank correlation is more appropriate:

rho, p_value = stats.spearmanr(season_2015['salary'], season_2015['OPS'])

print(f'Spearman rho: {rho:.3f}')
print(f'p-value: {p_value:.4f}')

Spearman’s ρ (rho) works by ranking both variables and computing the Pearson correlation on the ranks. It is robust to outliers and does not assume the relationship is linear.

18.8 Correlation Across Many Variables

When you have several numeric columns, compute the full pairwise correlation matrix and identify which pairs have the strongest relationships:

metrics = bat_sal[['AB', 'HR', 'BB', 'BA', 'SLG', 'OBP', 'OPS', 'salary']].dropna()
# .corr() computes Pearson correlation between every pair of columns
# Values range from -1 (perfect negative) to +1 (perfect positive), 0 = no linear relationship
corr = metrics.corr()

fig, ax = plt.subplots(figsize=(8, 6))
sns.heatmap(
    corr,
    annot=True,      # print the r value inside each cell
    fmt='.2f',       # round to 2 decimal places
    cmap='coolwarm', # red = positive correlation, blue = negative, white = near zero
    center=0,        # anchor the neutral color at r=0
    square=True,
    linewidths=0.5,
    ax=ax
)
ax.set_title('Correlation Matrix — Batting Metrics and Salary')
plt.tight_layout()
plt.show()

Look for the cells with the strongest colors. Correlations between derived metrics (like OBP and OPS) will be high by construction. The interesting correlations are between conceptually distinct variables — salary and performance metrics, counting stats and rate stats.

18.9 Summary: Choosing a Test

Question Test Input
Is the mean different between two independent groups? Independent t-test (stats.ttest_ind) One numeric + one binary categorical
Is there a systematic difference between two paired measurements? Paired t-test (stats.ttest_rel) Two numeric columns from the same entity
Are two categorical variables associated? Chi-square test (stats.chi2_contingency) Two categorical variables (contingency table)
How strong is the linear relationship between two variables? Pearson correlation (stats.pearsonr) Two numeric variables
How strong is the monotonic (but possibly non-linear) relationship? Spearman correlation (stats.spearmanr) Two numeric variables

18.10 Reporting Results

When reporting the result of a statistical test, include four elements:

  1. What you tested: “We compared mean batting average between the pre-steroid (1980–93) and steroid (1994–2004) eras.”
  2. The result: “The steroid era had a slightly higher mean batting average (0.271 vs. 0.267).”
  3. The test statistic and p-value: “t(8452) = -3.21, p = 0.001”
  4. The effect size and interpretation: “Cohen’s d = 0.07, a small effect. While the difference is statistically significant due to the large sample size, the practical difference of 4 points of batting average is modest.”

This pattern — what, result, statistics, interpretation — keeps your reporting honest and complete. It prevents the common mistake of reporting a p-value without context about the magnitude of the effect.

```{python}
    (qualified['yearID'] >= 1980) & (qualified['yearID'] <= 1993)
]['BA'].dropna()

steroid_era = qualified[
    (qualified['yearID'] >= 1994) & (qualified['yearID'] <= 2004)
]['BA'].dropna()

print(f'Pre-steroid: n={len(pre_steroid)}, mean={pre_steroid.mean():.4f}, std={pre_steroid.std():.4f}')
print(f'Steroid era: n={len(steroid_era)}, mean={steroid_era.mean():.4f}, std={steroid_era.std():.4f}')
```

Visualize the distributions before testing:

```{python}
fig, ax = plt.subplots(figsize=(8, 4))
sns.kdeplot(pre_steroid, label='Pre-steroid (1980–93)', fill=True, alpha=0.3, ax=ax)
sns.kdeplot(steroid_era, label='Steroid era (1994–2004)', fill=True, alpha=0.3, ax=ax)
ax.set_xlabel('Batting Average')
ax.set_title('Batting Average Distribution: Pre-Steroid vs. Steroid Era')
ax.legend()
plt.tight_layout()
plt.show()
```

Now run the t-test:

```{python}
# ttest_ind: independent-samples t-test (comparing two separate groups)
# Returns (t_statistic, p_value) as a tuple
# A large |t| and small p indicates the means are unlikely to be equal by chance
t_stat, p_value = stats.ttest_ind(pre_steroid, steroid_era)

print(f't-statistic: {t_stat:.3f}')
print(f'p-value: {p_value:.4f}')
```

### Interpreting the result

The mean difference between the two eras can be computed directly:

```{python}
diff = steroid_era.mean() - pre_steroid.mean()
print(f'Mean difference: {diff:.4f}')
```

Here is what you can conclude: if the p-value is below 0.05, the difference in mean batting average between the two eras is statistically significant — unlikely to have arisen by chance. The effect size (the mean difference) tells you whether the difference is large enough to be practically meaningful. A difference of 0.005 in batting average is detectable with thousands of player-seasons but may not be substantively interesting. A difference of 0.015 would be more notable.

What you **cannot** conclude: that steroids caused the difference. These are different groups of players measured in different years with different rules, ballparks, and training methods. The comparison is between-subjects and observational — all the caveats from @sec-research-questions apply.

### Effect size: Cohen's d

Cohen's d measures the difference in means in terms of standard deviations. It provides a standardized measure of effect size that is comparable across different variables and studies.

```{python}
# Cohen's d = mean difference / pooled standard deviation
# Pooled std averages the variance of both groups, giving equal weight to each
pooled_std = np.sqrt((pre_steroid.std()**2 + steroid_era.std()**2) / 2)
cohens_d = (steroid_era.mean() - pre_steroid.mean()) / pooled_std

# Interpretation guide: |d| < 0.2 = negligible, 0.2–0.5 = small, 0.5–0.8 = medium, > 0.8 = large
print(f"Cohen's d: {cohens_d:.3f}")
```

Rules of thumb for interpreting Cohen's d: 0.2 is a small effect, 0.5 is medium, and 0.8 is large. These are guidelines, not thresholds — context determines what counts as meaningful in your domain.

## Comparing Two Related Measurements: The Paired t-Test

A paired t-test is appropriate when you have two measurements from the same entity — for example, medical and retail cannabis sales from the same county in the same month. The question is: *Is there a systematic difference between the two measurements?*

```{python}
# Compare medical vs retail sales for county-months where both exist
paired = cannabis[cannabis['medical'].notna() & cannabis['retail'].notna()].copy()

print(f'Paired observations: {len(paired)}')
print(f'Mean medical: ${paired["medical"].mean():,.0f}')
print(f'Mean retail:  ${paired["retail"].mean():,.0f}')

t_stat, p_value = stats.ttest_rel(paired['retail'], paired['medical'])

print(f'\nPaired t-test:')
print(f't-statistic: {t_stat:.3f}')
print(f'p-value: {p_value:.6f}')
```

Here is what you can conclude: if the p-value is very small, retail sales are systematically different from medical sales across county-months. The direction and magnitude of the difference tell you which is larger and by how much. This is a within-subjects comparison — the same counties are measured on both variables — so many county-level confounds (population, geography) are controlled for.

## Comparing Categorical Variables: The Chi-Square Test

The chi-square test of independence asks: *Are two categorical variables associated, or are they independent?* Use it when both your variables are categorical.

### Example: Has the distribution of home run tiers changed across eras?

```{python}
# Bin HR into categories
hr_data = qualified[qualified['decade'].isin([1980, 2000])].copy()
hr_data['hr_tier'] = pd.cut(
    hr_data['HR'], bins=[-1, 5, 15, 30, 100],
    labels=['0-5', '6-15', '16-30', '31+']
)
hr_data['era'] = hr_data['decade'].map({1980: '1980s', 2000: '2000s'})

# Build contingency table
contingency = pd.crosstab(hr_data['era'], hr_data['hr_tier'])
print('Contingency table:')
print(contingency)
```

```{python}
chi2, p_value, dof, expected = stats.chi2_contingency(contingency)

print(f'\nChi-square statistic: {chi2:.2f}')
print(f'Degrees of freedom: {dof}')
print(f'p-value: {p_value:.6f}')
```

Here is what you can conclude: if the p-value is small, the distribution of home run tiers differs significantly between the two eras — the two variables (era and HR tier) are not independent. To understand *how* they differ, compare the observed counts to the expected counts (what you would see if era and HR tier were independent):

```{python}
expected_df = pd.DataFrame(
    expected, index=contingency.index, columns=contingency.columns
).round(1)

print('Expected counts (if independent):')
print(expected_df)
print('\nObserved - Expected:')
print((contingency - expected_df).round(1))
```

Positive residuals indicate cells with more observations than expected; negative residuals indicate fewer. This tells you which specific categories drive the overall association.

### Effect size: Cramér's V

Cramér's V is the chi-square analog of Cohen's d — a measure of how strongly the two categorical variables are associated, scaled from 0 (independent) to 1 (perfectly associated):

```{python}
n = contingency.sum().sum()
min_dim = min(contingency.shape) - 1
cramers_v = np.sqrt(chi2 / (n * min_dim))

print(f"Cramér's V: {cramers_v:.3f}")
```

## Measuring Linear Relationships: Correlation

The Pearson correlation coefficient (r) measures the strength and direction of a linear relationship between two numeric variables. It ranges from -1 (perfect negative linear relationship) to +1 (perfect positive linear relationship), with 0 indicating no linear relationship.

### Example: Salary and performance

```{python}
# Filter to a single recent season with enough data
season_2015 = bat_sal[bat_sal['yearID'] == 2015].dropna(subset=['salary', 'OPS'])

r, p_value = stats.pearsonr(season_2015['salary'], season_2015['OPS'])

print(f'Pearson r: {r:.3f}')
print(f'p-value: {p_value:.4f}')
print(f'n: {len(season_2015)}')
```

Visualize alongside the statistic:

```{python}
fig, ax = plt.subplots(figsize=(8, 5))
ax.scatter(season_2015['salary'] / 1e6, season_2015['OPS'], alpha=0.4, s=20)
ax.set_xlabel('Salary (millions $)')
ax.set_ylabel('OPS')
ax.set_title(f'Salary vs. OPS (2015) — r = {r:.3f}, p = {p_value:.4f}')
plt.tight_layout()
plt.show()
```

Here is what you can conclude: the correlation coefficient tells you the strength and direction of the linear relationship. A positive r means higher salary is associated with higher OPS. The p-value tells you whether this correlation is statistically significant. The magnitude of r tells you how tightly the points cluster around a line — an r of 0.3 indicates a weak-to-moderate positive relationship, meaning salary explains about 9% (r² = 0.09) of the variation in OPS.

What you cannot conclude: that paying a player more would improve their performance. As discussed in @sec-research-questions, the causal arrow is ambiguous, and the relationship is confounded by age, experience, and roster construction.

### Spearman correlation for non-linear monotonic relationships

Pearson's r measures linear relationships. If the relationship is monotonic but curved (both variables increase together, but not in a straight line), Spearman's rank correlation is more appropriate:

```{python}
rho, p_value = stats.spearmanr(season_2015['salary'], season_2015['OPS'])

print(f'Spearman rho: {rho:.3f}')
print(f'p-value: {p_value:.4f}')
```

Spearman's ρ (rho) works by ranking both variables and computing the Pearson correlation on the ranks. It is robust to outliers and does not assume the relationship is linear.

## Correlation Across Many Variables

When you have several numeric columns, compute the full pairwise correlation matrix and identify which pairs have the strongest relationships:

```{python}
metrics = bat_sal[['AB', 'HR', 'BB', 'BA', 'SLG', 'OBP', 'OPS', 'salary']].dropna()
# .corr() computes Pearson correlation between every pair of columns
# Values range from -1 (perfect negative) to +1 (perfect positive), 0 = no linear relationship
corr = metrics.corr()

fig, ax = plt.subplots(figsize=(8, 6))
sns.heatmap(
    corr,
    annot=True,      # print the r value inside each cell
    fmt='.2f',       # round to 2 decimal places
    cmap='coolwarm', # red = positive correlation, blue = negative, white = near zero
    center=0,        # anchor the neutral color at r=0
    square=True,
    linewidths=0.5,
    ax=ax
)
ax.set_title('Correlation Matrix — Batting Metrics and Salary')
plt.tight_layout()
plt.show()
```

Look for the cells with the strongest colors. Correlations between derived metrics (like OBP and OPS) will be high by construction. The interesting correlations are between conceptually distinct variables — salary and performance metrics, counting stats and rate stats.

## Summary: Choosing a Test

| Question | Test | Input |
|----------|------|-------|
| Is the mean different between two independent groups? | Independent t-test (`stats.ttest_ind`) | One numeric + one binary categorical |
| Is there a systematic difference between two paired measurements? | Paired t-test (`stats.ttest_rel`) | Two numeric columns from the same entity |
| Are two categorical variables associated? | Chi-square test (`stats.chi2_contingency`) | Two categorical variables (contingency table) |
| How strong is the linear relationship between two variables? | Pearson correlation (`stats.pearsonr`) | Two numeric variables |
| How strong is the monotonic (but possibly non-linear) relationship? | Spearman correlation (`stats.spearmanr`) | Two numeric variables |

## Reporting Results

When reporting the result of a statistical test, include four elements:

1. **What you tested:** "We compared mean batting average between the pre-steroid (1980–93) and steroid (1994–2004) eras."
2. **The result:** "The steroid era had a slightly higher mean batting average (0.271 vs. 0.267)."
3. **The test statistic and p-value:** "t(8452) = -3.21, p = 0.001"
4. **The effect size and interpretation:** "Cohen's d = 0.07, a small effect. While the difference is statistically significant due to the large sample size, the practical difference of 4 points of batting average is modest."

This pattern — what, result, statistics, interpretation — keeps your reporting honest and complete. It prevents the common mistake of reporting a p-value without context about the magnitude of the effect.
